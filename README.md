# 💰 Controle de Despesas

Um projeto desenvolvido para praticar os fundamentos do desenvolvimento Front-End, permitindo registrar e acompanhar despesas de forma simples e intuitiva.

## 📖 Sobre o projeto

O **Controle de Despesas** foi criado com o objetivo de aplicar conceitos de HTML, CSS e JavaScript em uma aplicação real, simulando um sistema básico de gerenciamento financeiro.

Com ele é possível cadastrar despesas, visualizar os registros e acompanhar os gastos de maneira organizada.

## ✨ Funcionalidades

* ➕ Adicionar novas despesas
* 📋 Listar todas as despesas cadastradas
* 💲 Exibir o valor de cada despesa
* 🧮 Calcular o total de despesas
* 🗑️ Remover despesas cadastradas
* 💾 Armazenamento local utilizando **LocalStorage**, mantendo os dados mesmo após fechar o navegador.

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)
* LocalStorage

## 🚀 Como executar

1. Clone este repositório.
2. Abra a pasta do projeto.
3. Execute o arquivo `index.html` em seu navegador.
